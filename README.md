# DRUM TEMPO ACCELERATOR
〜ドラム練習の限界を突破する、自動テンポ加速メトロノーム〜

ドラムの基礎練習において「テンポを徐々に上げていく練習」は技術と持久力の向上に不可欠です。しかし、テンポを変えるたびにスティックを置き、メトロノームを操作し直すのは集中力を削ぐ原因になります。

「DRUM TEMPO ACCELERATOR」は、そのテンポアップ作業を全自動化し、ドラマーがスティックを一切置くことなくドラムパッドやセットに向き合い続けるために開発されました。

---

## ⚡ 主な特徴 (Features)

### 1. 全自動テンポアップシステム
「開始BPM」「目標（限界）BPM」「加速する間隔（小節数）」「1回あたりの加速量（+BPM）」を自由に設定可能。あなたの設定通りに、練習の進行に合わせてテンポがシームレスに上昇していきます。

### 2. ワンタップ・プリセット保存機能
「パラディドル限界突破（BPM 100 ➔ 160）」「シングルストローク持久力強化（BPM 120 ➔ 180）」など、用途に合わせた練習メニューを名前をつけて個人アカウントに保存可能。練習開始の準備時間をゼロにします。

### 3. ドラマーに最適化された「スクロール不要」の超コンパクトUI
自動加速モードのすべての入力フォームと保存ボタンを全展開しても、スタート/ストップボタンが画面内にピッタリと収まるタイトな設計。ドラムセットにスマートフォンを置いた状態でも、抜群の視認性と操作性を誇ります。

### 4. 練習のモチベーションを高める「ヨルシカ風」の美しいデザイン
深みのある藍色の夜空から、茜色の夕暮れ、月光のような儚いイエローを基調とした、プレミアム感漂うガラスモフィズム（透過ガラス）デザイン。日々の退屈になりがちな基礎練習を、美しい没入空間へと変えます。

### 5. 達成感を刺激するアチーブメント演出
設定した「目標（限界）BPM」に到達した瞬間、あなたの努力を称えるアチーブメント画面がポップアップ。日々のモチベーション維持と上達を視覚的にサポートします。

## 🥁 こんなドラマーにおすすめ
- ダブルストロークやパラディドルのスピードアップ、ゴーストノートの精度向上を目指す方。
- 手足のコンビネーションなど、特定のフレーズを徐々にテンポアップして体に馴染ませたい方。
- メトロノームを毎回触る時間を削り、1秒でも多くドラムを叩く練習時間に充てたい方。

## 📂 プロジェクト構成

本リポジトリは以下の2種類のフロントエンドおよびバックエンドAPIサーバーで構成されています。

```text
drum_prototype/
├── index.html       # 1) 超軽量・バニラJavaScript版（HTML単一ファイルで稼働可能）
├── frontend/        # 2) Next.js / TypeScript / React版（プロダクション品質）
└── backend/         # 3) Spring Boot / MyBatis / H2データベース（プリセット用API）
```

---

## 🚀 起動・開発手順

### 1) バニラJS版 (index.html) の実行
`index.html` ファイルを任意のブラウザで直接開くだけで起動します。ビルドやローカルサーバーの立ち上げは不要です。
※ プリセットの保存機能を利用する場合は、下記のバックエンドAPIを起動した状態でログインしてください。

### 2) Next.js版 (frontend) の起動
```bash
cd frontend
npm install
npm run dev
```
起動後、ブラウザで [http://localhost:3000](http://localhost:3000) にアクセスします。

### 3) APIサーバー (backend) の起動
Java 17 (Maven) がインストールされている環境で以下を実行します。
```bash
cd backend
mvn spring-boot:run
```
H2データベースを利用し、ポート `8080` でログイン・プリセットAPIが稼働します。
